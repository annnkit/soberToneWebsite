# SoberTone Website Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Vibrant purple (#A47DF2) - representing innovation and transformation
- **Secondary**: Electric blue (#3BCDF7) - representing technology and trust
- **Accent**: Light cyan (#D0F5FF) - representing clarity and openness
- **Dark**: Deep navy (#032A5C) - representing stability and depth
- **Text**: Deep navy (#032A5C) - ensuring 4.5:1 contrast ratio

### Typography
- **Display Font**: "Playfair Display" - elegant serif for headings and hero text
- **Body Font**: "Inter" - clean, readable sans-serif for content
- **Accent Font**: "Source Code Pro" - monospace for technical elements

### Visual Language
- **Minimalist Approach**: Clean, uncluttered design focusing on content
- **Organic Shapes**: Soft, rounded elements representing healing and comfort
- **Breathing Space**: Generous whitespace for mental clarity
- **Gentle Gradients**: Subtle transitions between colors, never harsh

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: For smooth micro-interactions and element animations
- **Typed.js**: For typewriter effects on key statistics and messaging
- **ECharts.js**: For data visualizations and impact calculators
- **p5.js**: For creative coding elements and voice waveform visualizations
- **Splide.js**: For team member carousel and testimonials
- **Matter.js**: For subtle physics-based animations on floating elements

### Animation Effects
- **Typewriter Animation**: Key statistics appear with typing effect
- **Fade-in Stagger**: Content sections reveal with staggered timing
- **Floating Elements**: Subtle movement on decorative shapes
- **Voice Waveforms**: Real-time visualization during AI demo
- **Progress Animations**: Smooth transitions for impact calculations

### Header Effect
- **Gradient Background**: Soft teal to sage gradient with animated particles
- **Parallax Scrolling**: Subtle depth effect on hero elements
- **Mascot Integration**: SoberTone mascot integrated into hero section
- **Interactive Voice Demo**: Central interactive element in hero

### Interactive Elements
- **Hover Transformations**: 3D tilt effects on cards and buttons
- **Color Transitions**: Smooth color changes on interactive elements
- **Shadow Depth**: Dynamic shadows that respond to user interaction
- **Ripple Effects**: Subtle feedback on clickable elements

### Responsive Design
- **Mobile-First**: Optimized for mobile experience
- **Flexible Grid**: CSS Grid and Flexbox for responsive layouts
- **Scalable Typography**: Fluid type scale across devices
- **Touch-Friendly**: Appropriate touch targets for mobile interaction
